### About me

I'm a software engineer at **Nextuple**, working on order management systems — think SQL, queue monitoring, and keeping enterprise data flows honest. Outside of work, I'm building a macOS-style portfolio in React, complete with a fully interactive terminal easter egg and a pixel-perfect cloned UI. Right now I'm also learning AI/ML from first principles — currently deep in how models actually learn.

### What I'm up to right now

- 🛠️ **Building:** a macOS-style portfolio in React + GSAP + Tailwind — with a hidden `HarshOS Terminal` easter egg (boot sequence, command history, tab autocomplete — try typing `help` when you find it) and a pixel-perfect Google SERP clone inside a Safari window component
- 🧠 **Learning:** AI/ML from scratch — how models store learned patterns as weights, what loss functions actually do, why labeled data is the real bottleneck
- ✍️ **Writing:** short, fact-heavy posts on cybersecurity and tech on LinkedIn
- 📫 **Reach me:** Hariharshvardhan22@gmail.com
