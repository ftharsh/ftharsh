### About me

Software engineer at **Nextuple**, shipping features and squashing bugs across enterprise order management systems — sprint planning, automation, CI/CD, the full cycle. Before that, I built an EdTech platform at PCS Global and a manufacturing optimization tool at Braithwaite & Co. Outside work, I build full-stack side projects, mess around with real-time systems, and I'm currently learning AI/ML from absolute scratch.

B.Tech in CSE from **Sikkim Manipal Institute of Technology** (9.2 GPA, Class of 2024).

---

### What I do

**🏢 Enterprise & Backend** — Working on order management systems at Nextuple across 4 projects (2 client-facing, 2 internal tools). Closed 100+ tickets in my first 4 months. Collaborated on 30+ automation scenarios, boosting test coverage by 35%.

**⚛️ Full-Stack Products** — I build complete apps end-to-end: digital wallets handling 1000+ txns/day with Kafka + SMTP alerts, real-time multiplayer chess with <50ms latency via Socket.io, a full Excalidraw clone with 10+ drawing tools, and an EdTech platform that improved task completion by 25%.

**🧠 AI/ML (Learning)** — Currently studying from first principles: how weights and biases store learned patterns, what loss functions actually optimize, why labeled data is the real bottleneck. Early days, but building up deliberately.

**🎨 Portfolio & Creative** — Building a macOS-style portfolio site in React + GSAP + Tailwind with a hidden **HarshOS Terminal** easter egg — complete with boot sequence, tab autocomplete, and command history. There's also a pixel-perfect Google SERP clone inside a Safari window component. Not deployed yet, but it's my most polished frontend work.
